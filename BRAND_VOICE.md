# Brewmie voice

Punchy, second-person, anti-ceremony. The user is a competent home barista. Don't explain the obvious, don't oversell what the app does, don't reach for "AI / smart / intelligent / powered by / algorithm / your journey". No em dashes; use periods or rewrite. Lower-case is fine when rhythm calls for it. Verbs over nouns, fragments over full sentences. Reasons should land in one beat: cause, then call. Lazy Sous is the sister brand; same studio, same respect for the user's time.

Say this, not that:
- "Ran 5s long. Go coarser." not "Your shot ran 5 seconds long, so we recommend going slightly coarser."
- "Pull a shot." not "Ready to dial in?"
- "Your kit, on file." not "Your machine and grinder shape every shot. Set them up once and we'll remember."
